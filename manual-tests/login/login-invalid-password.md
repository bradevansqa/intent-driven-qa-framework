# Login - Invalid Password

## Intent
Verify that a registered user cannot log in with an incorrect password and receives a clear, actionable error message.

## Feature
Authentication

## Page / Area
Login

## User Type
Registered user

## Risk Areas
- Authentication logic
- Error messaging
- Security (credential handling)
- User experience

## Preconditions
- User account exists
- User is logged out

## Expected Outcome
- Login is rejected
- User remains on the login page
- A clear error message is displayed

## Automation Status
Planned

## Notes
High-frequency regression risk when authentication or validation logic changes.
